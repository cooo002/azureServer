# 2019 PBL 사업을 위한 프로젝트
발생한 위험 상황을 직접 볼 수 없는 시각장애인을 위하여 시각장애인이 **음성인식**을 통해 촬영된 상황을 마이크로소프트 Azure Cloud를 사용하여 
분석하고 분석된 상황을 음성으로 시각장애인에게 제공하는 서비스(_본 기능 이외에 재난경보 알리미와 같은 부수적인 기능도 수행_)
